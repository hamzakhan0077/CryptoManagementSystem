MUST HAVE VIRTUAL Environment(venv) TO RUN THE DEVELOPER SERVER

-There is already a virtual environment in CryptoSystem root directory called cryptovenv

-OPEN CMD navigate to project directory

-cd into cryptovenv/Scripts

-run the activate using the command
.\activate

-you are now in virtual environment
-your cmd will be like this

-(cryptovenv)C:// blah/blah/blah path
-now navigate back to CryptoSystem directory

-where run.py file is

-simply type in the command

-python run.py

-THe server will start at localhost copy and past the url in the browser.
